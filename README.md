# NEMU
A simple NES emulator created in c++
![Preview](https://raw.githubusercontent.com/Cherrytree56567/NEMU/refs/heads/master/preview.png)

## Building
For Windows run:
```
mkdir build
cd build
cmake ..
cmake --build .
```

## Features
 - NROM Mapper
